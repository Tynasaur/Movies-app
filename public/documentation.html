Movies Index

Documentation

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

URL Endpoints

Business Logic URL HTTP Method Request Response
Return a list of ALL movies /movies GET None A JSON object holding data about movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:Title GET None A JSON object holding the description, genre, director, image URL, whether it’s featured or not of a movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /genre/:Name GET None A JSON obect holding the genre of the movie
Return data about a director (bio, birth year, death year) by name /director/:Name GET None A JSON holding the director of a movie by name
Allow new users to register /users/:Username POST A JSON object holding data about the new user, structured like: { name: 'Johnny' username: '[JohnnyRocks]' password: '[password]', firstName: '[firstName]', lastName: '[lastName]' } A JSON object holding data about the user that was added, including username { name: 'Johnny' username: '[JohnnyRocks]' password: '[password]', firstName: '[firstName]', lastName: '[lastName]' }
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/:Username/:Title PUT None A text message indicating movie was added to their favorites list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:Username/:Title DELETE None A text message indicating movie was succesfully removed from list of favorites
Allow existing users to deregister (showing only a text that a user email has been removed) /users/:Username DELETE None A text message indicating user was deregistered
