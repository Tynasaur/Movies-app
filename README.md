# myFlixAPI

This is a movie API that can be used to access a list of movies and movie details: Title, Genre, Director and Description.


## Tools Used

**Languages:**
  JavaScript

**Database:**
  MongoDB

**Framework:**
  Express
  
**Server Environment:**
  Node.js
  
  
  ## Dependencies
  
  * express
  
  * mongoose
  
  * morgan
  
  * passport
  
  * cors
